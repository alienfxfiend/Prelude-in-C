For use /w AutoHotkey 1.1 (needs 1.1 to be installed (not v2)) ~~> https://www.autohotkey.com/download/ahk-install.exe
Edit .AHK in TextEditor & change "filepath" to the path of your SumatraPDF Settings TextFile
Double-click entry or select & hit Apply. Escape to close.
Shows count of themes configured.
Default goes back to Factory Default Black & White Out-Of-Box *Theme* reset (no changes to background color in illustrated e-Books)
